설치 방법
  1. zip 파일을 다운로드 후 원하는 폴더에 압축 해제합니다.
  2. 폴더 안의 .exe 파일을 더블클릭해 실행합니다. (폴더 통째로 보관 — _internal 폴더가 옆에 있어야 함)
  3. 처음 실행 시 Windows SmartScreen 경고가 뜨면 추가 정보 → 실행을 누르세요. (서명되지 않은 사내 빌드라 정상입니다.)
